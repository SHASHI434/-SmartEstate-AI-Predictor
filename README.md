# 🏠 SmartEstate AI Predictor

An AI-powered web application that predicts house prices based on property features like area, number of rooms, age, and distance from the city.

---

## 🌐 Live Demo

🚀 **Deployed App:**
👉 *https://smartestate-ai-predictor.streamlit.app/*

---

## 📌 About the Project

SmartEstate AI Predictor is a Machine Learning-based application that helps users estimate house prices using real-world features. It uses an optimized regression model to provide fast and accurate predictions.

---

## 🚀 Features

* 📊 Predict house prices instantly
* 🧠 Machine Learning model (XGBoost)
* 🌐 Interactive UI using Streamlit
* 📝 Stores user prediction history
* ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas
* Scikit-learn
* XGBoost
* Streamlit
